## mi delete

Delete users from a Micro Integrator instance

### Synopsis

Delete users from a Micro Integrator instance in the environment specified by the flag (--environment, -e)

```
mi delete [flags]
```

### Examples

```
 mi delete user capp-tester -e dev
```
